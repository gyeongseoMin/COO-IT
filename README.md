# 모바일프로그래밍 4팀 프로젝트 'COO-IT'
#### 냉장고 속 재료로 만들 수 있는 요리 레시피 추천 앱
## Introduction
1. 자신의 냉장고에 있는 재료로 만들 수 있는 레시피를 추천해준다.
2. 상세레시피를 통해 자세한 요리 방법과 사진들을 볼 수 있다. 
3. 카테고리별로 나누어 음식을 추천해준다.
4. 레시피를 스크랩할 수 있다.
